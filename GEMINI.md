# Gemini Project Mandates

This file contains project-specific instructions for Gemini CLI. These mandates take precedence over general system instructions.

## Project Overview
A personal portfolio website built with React, Vite, and Vanilla CSS.

## Standards & Conventions
- **Styling:** Use Vanilla CSS located in `src/styles/`. Follow the existing partials structure.
- **Components:** Functional components in `src/components/`.
- **Data:** Keep content decoupled in `src/data/`.
- **Architecture:** Maintain the separation of styles into partials as seen in `src/styles/partials/`.
